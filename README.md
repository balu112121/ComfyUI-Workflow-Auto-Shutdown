# ComfyUI 工作流自动关机插件小助手

🤖 一个智能的ComfyUI插件，在工作流完成后自动保存结果并关机，节省电力和时间。

## 功能特点

- ✅ 工作流完成后自动关机
- ✅ 自动保存生成的图像和视频
- ✅ 可自定义关机延迟时间
- ✅ 支持取消关机计划
- ✅ 完整的中文界面
- ✅ 跨平台支持（Windows/macOS/Linux）

## 安装方法

1. 将整个 `ComfyUI-Workflow-Auto-Shutdown` 文件夹复制到 `ComfyUI/custom_nodes/` 目录下
2. 重启ComfyUI
3. 在节点菜单中找到"⚡ 自动关机小助手"类别

## 节点介绍

### 1. 工作流自动关机小助手

**输入参数:**
- `启用自动关机`: 是否启用自动关机功能
- `关机延迟时间`: 工作流完成后多少秒关机（10-600秒）
- `保存输出文件`: 是否保存输出的图像和视频
- `输出目录`: 文件保存的目录名称
- `保存图像`: 是否保存图像文件
- `保存视频`: 是否保存视频文件
- `图像输入`: 连接的图像输入（可选）
- `视频输入`: 连接的视频输入（可选）

**输出:**
- `图像输出`: 传递输入的图像
- `视频输出`: 传递输入的视频

### 2. 取消计划关机

**输入参数:**
- `确认取消`: 确认取消关机计划

## 使用教程

### 基础使用

1. 在工作流最后添加"工作流自动关机小助手"节点
2. 连接图像和视频输出到该节点
3. 配置关机参数：
   - 设置关机延迟时间（建议60秒以上）
   - 选择是否保存文件
   - 设置输出目录
4. 运行工作流，完成后会自动关机和保存文件

### 高级用法

- 可以连接多个输出到关机节点
- 使用"取消计划关机"节点随时取消关机
- 可以设置不同的输出目录来组织文件

## 注意事项

⚠️ **重要提示:**

1. 确保ComfyUI有执行关机命令的权限
2. 关机前请确保所有工作已保存
3. 建议先测试保存功能，再启用关机
4. 关机倒计时期间可以使用取消节点停止关机

## 故障排除

**Q: 关机命令执行失败？**
A: 可能需要以管理员权限运行ComfyUI

**Q: 文件没有保存？**
A: 检查输出目录权限和磁盘空间

**Q: 如何立即停止关机？**
A: 使用"取消计划关机"节点或关闭ComfyUI
